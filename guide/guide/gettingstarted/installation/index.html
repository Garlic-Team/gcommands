# Installing Node.js and GCommands

# Installing Node.js

Don't have Node.js yet? Go to NodeJS.org (opens new window) and install it! Don't have an editor? Check out Visual Studio Code (opens new window).

# Installing on Windows

If you have a Windows OS, it's as simple as installing any other program. Go to the Node.js website (opens new window), download the latest version, open the download file, and follow the steps in the installer.

# Installing on macOS

If you have macOS, you have a two options. You can:

  • go to the Node.js website (opens new window), download the latest version, open the download file, and follow the steps in the installer.
  • use a package manager like Homebrew (opens new window)

# Installing on Linux

If you have a Linux OS, you should go to this page (opens new window), to determine how to install Node.

Note

Make sure you upgrade to the needed Node.js version for discord.js.

  • discord.js@12 requires NodeJS@12+
  • discord.js@13 requires NodeJS@16+

# Installing GCommands

Use npm i gcommands to download the latest stable version.
Use npm i gcommands@dev to install the dev version.
